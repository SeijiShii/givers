# 寄付者による税額控除等の恩恵 — 必要な仕組み（検討）

寄付者が寄付により税額控除などの恩恵を受けるために、どのような仕組みが必要かを整理する。**日本で寄付の文化を根付かせることを第一目標**とするため（idea.md）、**日本での寄付に必要な情報の提供を最優先**し、本ドキュメントは日本（寄付金控除・認定NPO法人等・領収書）を中心に記載する。他法域は概要のみ。

**最終更新**: 2025-02

---

## 0. 前提：プラットフォームの役割

- **GIVErS プラットフォームは寄付金の授受を仲立ちしない。** 寄付金は寄付者から Stripe Connect を通じてプロジェクトオーナーに直接入金され、プラットフォームは入金の経由地とならない（idea.md に記載）。
- **税額控除の対象となる NPO 法人（認定NPO法人等）の場合のみ**、領収書等の書類発行を**手助けする**仕組みをプラットフォームが提供する。書類の発行主体はあくまで寄付金を受け取ったオーナー（団体）であり、プラットフォームは発行しない。

---

## 1. 日本の寄付金控除（個人）で必要なこと

### 1.1 制度の要点

- **対象**: 個人が**認定NPO法人等**（認定NPO法人、公益社団・財団法人、など）に寄付した場合、所得税の**寄付金控除**（所得控除または税額控除）を受けられる。
- **領収書**: 確定申告時に**領収書の添付**が求められる。領収書には**金額・年月日・寄附者の氏名および住所**が記載されている必要がある。
- **入金先**: 税制上の「寄付の相手」は**実際に寄付金を受け取る団体**。GIVErS ではオーナーに直接入金するため、**受け取り主体はプロジェクトオーナー（またはその代表団体）**であり、領収書もその団体が発行する。

### 1.2 GIVErS で提供する手助け（税額控除対象 NPO の場合のみ）

プラットフォームは**書類を発行しない**。認定NPO法人等であるオーナーが領収書等を発行する際の**手助け**として、次の仕組みを検討する。

| 手助けの内容 | 説明 |
|--------------|------|
| **「税額控除対象」の表明表示** | オーナーが「受け取り団体は認定NPO法人等です」と表明し、根拠（認定番号等）を登録したプロジェクトに限り、バッジ・ラベルで寄付者に表示する。 |
| **寄付者氏名・住所の取得** | 領収書に氏名・住所が必要なため、税額控除を希望する寄付では寄付時に入力（またはアカウント住所）を保存し、オーナーが参照できるようにする。 |
| **オーナー向け寄付一覧・エクスポート** | 自分のプロジェクトへの寄付について、金額・日付・寄付者氏名・住所を一覧・CSV/PDF で取得できる機能。オーナー（団体）が領収書を発行する際の根拠データとして利用する。 |
| **データの保持** | 寄付履歴は idea.md 方針で永続保存する。オーナー向けに「領収書用の寄付一覧」を出せるようにしておく。 |

### 1.3 運用の前提

- プラットフォームは寄付金を仲立ちしないため、領収書は**常に受け取ったオーナー（団体）が発行**する。GIVErS が提供するのは、**税額控除対象となる NPO の場合のみ**、上記の「表明表示・氏名・住所の取得・寄付一覧エクスポート」による**書類発行の手助け**である。
- 「寄付をいったんプラットフォームが受け取り、助成する」形は取らない（その場合は入金フロー・会計が変わり、プラットフォームが書類を発行する主体になるため、本方針の対象外）。

## 2. プラットフォーム側で検討・実装しうる機能（税額控除対象 NPO の場合のみ）

- **プロジェクトの「税額控除対象」フラグ**: オーナーが「受け取り団体は認定NPO法人等です」と表明し、根拠（認定番号等）を登録したプロジェクトに限り、詳細・寄付フォームに「寄付金控除の対象となります」等を表示。
- **寄付時の氏名・住所入力**: 領収書発行の手助けのため、税額控除を希望する寄付で氏名・住所を入力できる欄を設け、オーナー向け一覧に含める。
- **オーナー向け「寄付一覧・領収書用エクスポート」**: 自分のプロジェクトへの寄付について、金額・日付・寄付者氏名・住所を一覧・CSV/PDF で取得できる機能。オーナー（団体）が領収書を発行する際の手助けとする。
- **寄付者向け「領収書の発行依頼」**（任意）: 寄付者がオーナーに領収書発行を依頼し、オーナーに通知するフロー。発行はあくまでオーナー（団体）が行う。

## 3. 他法域（参考）

- **米国**: 501(c)(3) 等の免税団体への寄付で控除。団体が領収証を発行。プラットフォームが受け取る場合はプラットフォームの資格が重要。
- **英国**: Gift Aid の対象団体への寄付で税還付。団体が手続き。
- いずれも「**受け取り団体の資格**」と「**領収書・証明の仕組み**」が鍵。GIVErS が直接入金先であるか、オーナーに直接入金するかで、誰が領収書を発行するかが変わる。
